# gd-tools-flake
A flake wrapping https://codeberg.org/hashirama/gd-tools/, a fork of ajatt gd-tools. For use with Nix(os).

## for strokeorder to work: 
install `kanji-stroke-order-font` on the system (available in nix)

## for marisa to work: 
Put dictionary file `<name>.dic` into the `~/.local/share/gd-tools/` directory.  
(https://codeberg.org/hashirama/gd-tools/ has one for japanese in the res directory)
